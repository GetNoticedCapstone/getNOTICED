Project Description

getNoticed is a SaaS based web application that offers a individual an opportunity to have their own modern and mobile friendly web portfolio without the need of any programming or design skills. A web portfolio creates a centralized location that showcases a persons top skills and qualifications, making it a great option for a individual to set them self's apart other candidates. What makes our product great is that it has the capability to support an unlimited amount of users and a unlimited amount of websites that are all accessible through the web. In addition to this functionality our entire web based saas application incorporates responsive design making it mobile friendly on all devises.
